About Me

Who am I?

Hi, I'm Yashwanth Kumar — an MBA student and passionate Java Full Stack Developer with strong interest in Microservices Architecture and scalable web applications. I enjoy building modern, responsive, and user-friendly applications using Java, Spring Boot, ReactJS, Tailwind CSS, and Microservices.

I have worked on projects like CRM systems and e-commerce applications, implementing features such as authentication, REST APIs, dashboards, lead management, and responsive UI design. I also have hands-on experience with Spring Security, JWT Authentication, MySQL, Docker, Git, and Microservices concepts including Eureka Service Registry and Feign Client communication.

My MBA background helps me combine technical skills with business understanding to create practical and impactful solutions for real-world problems. Currently focused on becoming a skilled Java Full Stack Developer and building scalable Microservices-based applications for startups and businesses.
